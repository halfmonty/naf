NAF-HTML

HTML-First Reactive Bindings

What is NAF-HTML?

NAF-HTML is a ~1.1KB gzipped variant of NAF designed for progressive enhancement. Your HTML renders immediately, then JavaScript binds reactivity to existing elements.

Features

  • Reactive Signals - Fine-grained reactivity without a virtual DOM
  • Computed Values - Automatic dependency tracking and caching
  • Effects - Side effects that re-run when dependencies change
  • HTML-First - Markup renders before JS loads
  • Keyed Lists - Efficient list() with keyed updates
  • Two-Way Binding - model() for form inputs

Philosophy

HTML-first, not JS-first. Your content is visible immediately without waiting for JavaScript. NAF-HTML enhances existing markup with reactivity rather than generating it from scratch.
